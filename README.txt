Project requirements:
-JavaSE 1.8
-sunpkcs11.jar 	(located in /SEC)
-pteidlibj.jar	(located in /SEC)

To Run (Eclipse):

1: run FS-Server\src\fs\BlockServerJava
2: insert smart card
3: run FS-Client\src\main\App1 OR 
		FS-Client\src\main\App2

Notes:
App1 and App2 are programs that test some functionalities of the project.
For App1 you just need one smart card.
For App2 it's better to use two smart cards.
